## タイトル案
肌を撮るだけでケア提案まで。お金が切れても止まらない AI美容コンシェルジュを作った話

## 導入（何ができたか）
空のフォルダから始めて、GitHub・Vercel・AI API をつなぎ、  
**撮る → 提案 → ECへ誘導 → 毎日記録** まで動くWebツールを公開した。

公開URL: https://my-tool-bay.vercel.app

## なぜ作ったか
肌ケアの調べものが分散して面倒だった。  
「今日の肌に何をすればいいか」を、写真1枚で短く返してほしかった。

## どこまで作って、どこを捨てたか
### 残した
- 写真診断（AI + 無料ルールモード）
- スキンケア / メイク提案
- 化粧品のEC誘導
- 成長記録（文章のみ）

### 捨てた
- 医療診断
- 自前EC・決済
- 必須ログイン

## どう動くか（やさしく）
1. 同意する  
2. 肌写真を選ぶ  
3. 診断する  
4. 提案を見て、ECで商品を探す  
5. 今日の結果を記録する  

AI枠が切れたら、明るさ・赤みなどの簡易判定で提案を続ける。

## プライバシーでやったこと
- 同意必須  
- 写真はサーバーに残さない  
- 解析後プレビュー破棄  
- APIキーはサーバーのみ  

## つまずきと学び
- Vercel の Framework が Other のままだと静的配信で壊れる  
- OpenAI / Gemini は無料枠で止まる  
- キーをチャットに貼ると漏洩扱い  
- 「全部AI」より「止まらない設計」の方が大事  

## 使い方（最短）
1. https://my-tool-bay.vercel.app を開く  
2. プライバシーに同意  
3. 写真を選んで診断
